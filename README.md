# Project Mesto.Russia

https://mahakomar11.github.io/mesto/

### Description

Interactive site that allows to save photos from Russia with love (actually, with likes).

### Technologies

The following technologies were used:
1. Flexbox
1. Grid Layout
1. Media queries
1. BEM (Nested structure)
1. JavaScript
1. OOP
1. Webpack
1. Babel
1. PostCSS
1. API
